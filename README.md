## 📌 프로젝트 소개

### 차단 유저 필터링 시스템
- Lost Ark API 연동을 통한 유저 정보 자동 불러오기
- 차단 목록 캡처를 통한 자동 검별 시스템
- 파티 매칭 시 차단된 유저 자동 필터링
- 원정대 단위의 부캐릭터 확인 기능

- **배포**: Docker, AWS
